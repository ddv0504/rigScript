--- IMAGE PLANE FOLDER INSTRUCTIONS ---

Place images in this directory. Supported types...

PNG, JPG, JPEG, GIF, BMP, PBM, PGM, PPM, XBM, XPM

Refresh the Image Plane Tool
ZooToolsPro (shelf) > Camera Icon > Image Plane Tool > Dots (menu) > Refresh Thumbnails

Images will appear. 

Or change image directory with 
ZooToolsPro (shelf) > Camera Icon > Image Plane Tool > Dots (menu) > Set Image Dir



--- INCLUDED SAMPLE DIRECTORIES ---

Change folder to set the included sample directories. 
ZooToolsPro (shelf) > Camera Icon > Image Plane Tool > Dots (menu) > Set Image Dir...
